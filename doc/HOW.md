## AVL Tree

### Insert: (DONE)
- Search for the leaf node
	+ Go right
	+ Go left
- Find the right leaf node
- Add the node
- Go back to the root
- Rebalance the tree.

### Delete: (DONE)
- Search for the node
- Find the right leaf node
- Delete the node
- Rebalance the tree.

### Search: (DONE)
- Search for the node
- Find the right leaf node
